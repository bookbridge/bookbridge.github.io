PelicanのテーマをFlexに変更する

Posted on Mon 06 February 2017 in tech

Flexの導入

ダウンロード

% git clone https://github.com/alexandrevicenzi/Flex.git ~/themes/flex

必要に応じてプロジェクトサイトにthemes/をコピーする。

インストール

% pelican-themes -i themes/flex

pelicanconf.pyに以下を追記

THEME = 'themes/flex'

pelican

でThemeが適用されていることを確認出来ればOK

設定

こちらを参照

pelicanconf.py(抜粋)

AUTHOR = 'bookbridge'
SITENAME = 'bookbridge.github.io'
SITEURL = '.'
SITESUBTITLE = 'From Silicon to Clouds'
SITEDESCRIPTION = 'Thoughts and Writings'

PATH = 'content'
THEME = 'themes/flex'
TIMEZONE = 'Asia/Tokyo'

DEFAULT_LANG = 'ja'

SITELOGO = SITEURL + '/images/profile.jpg' #imagesはcontentの中に配置
