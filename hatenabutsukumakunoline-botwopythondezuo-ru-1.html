はてなブックマークのline botをpythonで作る (1)

Posted on 火 21 2月 2017 in tech

前提

  • python 3.6.0
  • Flask==0.12
  • line-bot-sdk==1.1.0

Line Developersに登録

  • Webhookを有効にしておく
  • Channel secretとChannel access tokenをメモる

Line command line sdkをインストール

$ pip install line-bot-sdk
$ git clone https://github.com/line/line-bot-sdk-python.git
$ cd example/flask-echo

Heroku SDKをインストール

https://devcenter.heroku.com/articles/heroku-cli

gunicornをインストール

$ pip install gunicorn

Procfileを作成

Procfile

web: gunicorn app:app --log-file -

Herokuにコミットする

$ git init
$ heroku create
$ git add --all
$ git commit -m `first commit'
$ git push heroku master

環境変数の設定

herokuで環境変数を節制するには以下の様にする。

$ heroku config:set LINE_CHANNEL_SECRET=<YOUR CHANNEL SECRET>
$ heroku config:set LINE_CHANNEL_ACCESS_TOKEN=<YOUR  TOKEN>

メッセージ送信してみる

lineでbotを友達に追加し、メッセージ送信してみる。 送信したメッセージがオウム返しで帰ってきたら正常動作です。

heroku line-bot flask
